# Sentiment Analyzer AI

A simple Flask web app that analyzes text sentiment.

## 🚀 Features
* Detects Positive, Negative, and Neutral sentiment
* Fast and easy to use

## 🛠 Tech Stack
Python (Flask), HTML

## ▶️ Run
```bash
pip install flask
python app.py
```

Open: http://127.0.0.1:5000
